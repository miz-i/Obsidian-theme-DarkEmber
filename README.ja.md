## 概要

- Obsidian 用 カスタムテーマ。

## 特徴

- Dark モード / Light モード。
- 見出しの装飾のオン/オフ。
- ４つのカラープリセット。
- [Style Settings Plugin](https://github.com/mgmeyers/obsidian-style-settings) 対応
- 今後、設定項目増やしていく予定。
